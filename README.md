# Building Agentic RAG with LlamaIndex

A repository for the **DeepLearning.AI** short course: **Building Agentic RAG with LlamaIndex**.

## Overview
Implementation of Agentic RAG systems using LlamaIndex, including:
- Router Query Engine
- Tool Calling / Function Calling
- Agent Reasoning Loop
- Multi-Document Agent

## Features
- Router-based query engines (QA + Summary)
- Tool calling with custom functions
- Multi-step reasoning agents
- Multi-document RAG agents
